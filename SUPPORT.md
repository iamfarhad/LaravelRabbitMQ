# LaravelRabbitMQ Support Policy

## Supported Versions

| PHP Version | Laravel Version | RabbitMQ Version     | Status          | End of Support |
|-------------|-----------------|----------------------|-----------------|----------------|
| 8.2         | ^11             | 3.8+                 | Active          | Dec 2026       |
| 8.3         | ^11 - ^12       | 3.8+                 | Active          | Dec 2027       |
| 8.4         | ^11 - ^13       | 3.13+                | Active          | TBA            |
| **8.5**     | ^12 - ^13       | **3.13+ / 4.x**      | **Active**      | TBA            |

## RabbitMQ Version Matrix

| RabbitMQ | Supported | Notes |
|----------|-----------|-------|
| 3.8 - 3.12 | ✅ Yes | Legacy support |
| **3.13**   | ✅ Yes | Recommended |
| **4.x**    | ✅ Yes | Full support with updated AMQP config |

**Security updates** will be provided for active versions only.
